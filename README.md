# English AR Learning — Restructured

โครงสร้างนี้ปรับให้ Part 1 และ Part 2 มีหน้าย่อยตามแผนการสอน

## โครงสร้างใหม่

```text
part1/
├── index.html
├── 1.1/index.html
├── 1.2/index.html
└── 1.3/index.html

part2/
├── index.html
├── 2.1/index.html
└── 2.2/index.html
```

- `part1/index.html` เป็นหน้าเมนูของ Part 1
- โค้ด Part 1 เดิมย้ายแนวคิดไปอยู่ที่ `part1/1.1/index.html`
- `part2/index.html` เป็นหน้าเมนูของ Part 2
- โค้ด Part 2 เดิมย้ายแนวคิดไปอยู่ที่ `part2/2.1/index.html`

เปิดใช้งานบน GitHub Pages ได้โดยอัปโหลดไฟล์ทั้งหมดนี้ขึ้น repository แล้วเปิด `index.html`
